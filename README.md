# GTAVPatcher
Script to automate GTA V patching process

Script must be placed on folder with all Patch subfolders:

Patch_1_0_0_1 (not a real patch, contains mpchristmas2, mpheist, patchday2bng, patchday3ng)  
Patch_1_0_323_1  
[...]  
Patch_1_0_791_2  
PatchGTAV.ps1
